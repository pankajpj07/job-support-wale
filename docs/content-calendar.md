## Adding a new post

1. Add a new entry to `blogPosts` in `src/data/blogPosts.ts` (slug, title, excerpt, category, date, author, readTimeMinutes, body).
2. Optionally add a featured image path to `image`.
3. Rebuild; the post will appear on `/blog` and in the RSS feed.
